# Axon Med Academy — Online Learning Platform

A production-ready online learning platform built for medical education, supporting live classes, recorded content, quizzes, and student management.

**Live Demo:** [axonmedacademy.com](https://axonmedacademy.com/)

## Overview

Axon Med Academy is a full-stack e-learning platform where instructors can create classes, manage students, host live sessions, and publish recorded lectures, quizzes, and announcements — all through a centralized admin dashboard.

## Features

- 📚 Class creation and management
- 🎥 Live classes with real-time join links
- 🎬 Recorded classes / on-demand video library
- 📢 Announcements system
- 📝 Quiz creation and student assessments
- 👩‍💻 Admin dashboard for managing students, classes, and content
- 🔐 Secure authentication for students and admins

## Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Auth:** JWT-based authentication
